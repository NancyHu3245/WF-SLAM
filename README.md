## Building 　　
```
cd ~/catkin_ws/src
git clone https://github.com/NancyHu3245/WF-SLAM.git
cd ..
./build.sh
```
## Usage
Configure the following four .json files in json-config in vscode, change the relevant file path according to your file path, and then press F5 to run.
